# Hoos Recycling

## Overview

Created recycling tracker for UVA for my Software Engineering class.

Some features include:
- Interactive map to add/remove recycling bins in certain locations
- User messaging
- Following other users
- Posting images

This project was created using Django and SQLite.

## How to Run Project

**Clone the repository**:
'''bash
git clone https://github.com/username/UVA-Recycling-App
Make sure you cd into the project folder: cd project-b-04
'''

**To build the virtual environment**

Create virtual environment using venv
'''bash
python -m venv venv
'''

Now activate
'''bash
Windows: venv\Scripts\activate
MacOS/Linux: source venv/bin/activate
'''

Make sure pip is installed
'''bash
pip install Django
'''

Verify and run:
'''bash
python manage.py runserver
'''

Now navigate to the URL: http://127.0.0.1
